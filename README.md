  特点
    1.解释型
        指代码不需要手动编译，而是通过解释器边编译边执行 把写完的代码直接给浏览器，一边编译一边执行 编译一句执行一句 即时编译
        运行JS，要安装JS的解释器（也叫JS引擎）
        浏览器自带解析器
        编译型
          计算机底层只认机器码（二进制码）
          编译型语言 写完代码通过编译转换成机器码，然后计算机执行机器码
